## Usage
Use when setting up visual regression testing via `/visual-regression`.

## Customization Points
- `{project-name}`: Project identifier
- `{port}`: Application port
- `scenarios`: Add pages/routes from your app
- `viewports`: Adjust breakpoints to match your design system
- `misMatchThreshold`: Pixel difference tolerance (0.1 = 0.1%)
- `cookiePath`: For authenticated pages

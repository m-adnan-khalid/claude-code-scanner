# Template: Playwright Configuration

Source: generated by `/e2e-browser` skill
Pattern: standard Playwright setup with multi-browser, screenshots, traces

```typescript
// playwright.config.ts
import { defineConfig, devices } from '@playwright/test';

export default defineConfig({
  testDir: './tests/e2e',
  fullyParallel: true,
  forbidOnly: !!process.env.CI,
  retries: process.env.CI ? 2 : 0,
  workers: process.env.CI ? 1 : undefined,
  reporter: [
    ['html', { open: 'never' }],
    ['json', { outputFile: '.claude/reports/e2e/results.json' }],
    ['junit', { outputFile: '.claude/reports/e2e/results.xml' }],
  ],
  use: {
    baseURL: process.env.BASE_URL || 'http://localhost:{port}',
    trace: 'on-first-retry',
    screenshot: 'only-on-failure',
    video: 'on-first-retry',
  },
  projects: [
    { name: 'chromium', use: { ...devices['Desktop Chrome'] } },
    { name: 'firefox', use: { ...devices['Desktop Firefox'] } },
    { name: 'webkit', use: { ...devices['Desktop Safari'] } },
    { name: 'mobile-chrome', use: { ...devices['Pixel 5'] } },
    { name: 'mobile-safari', use: { ...devices['iPhone 13'] } },
  ],
  webServer: {
    command: '{start-command}',
    url: 'http://localhost:{port}',
    reuseExistingServer: !process.env.CI,
    timeout: 30000,
  },
});
```

## Usage
Use when setting up Playwright for a new project via `/e2e-browser`.

## Customization Points
- `{port}`: Application port (e.g., 3000, 8000)
- `{start-command}`: Command to start the app (e.g., `npm run dev`)
- `testDir`: Path to E2E test files
- `projects`: Add/remove browsers as needed
- `webServer`: Configure for your app's startup

# Mobile Developer Profile

## Role
Mobile developer focused on iOS, Android, React Native, Flutter, or Kotlin Multiplatform apps.

## Primary Agents
- `@mobile` — your main development agent
- `@debugger` — for investigating mobile-specific issues
- `@tester` — for writing unit, widget, and integration tests
- `@code-quality` — for mobile architecture pattern review

## Key Skills
- `/mobile-audit` — comprehensive mobile quality audit
- `/workflow new "feature"` — full SDLC for mobile features

## Typical Workflow
```
/workflow new "mobile feature description"
# Phase 5 routes to @mobile automatically
```

## Focus Areas
- Screens, navigation, UI components
- State management (MVVM, BLoC, Redux)
- Offline-first data layer, caching, sync
- Platform APIs (camera, notifications, location, biometrics)
- App performance (startup time, memory, battery)
- Accessibility (VoiceOver, TalkBack, Dynamic Type)
- App store submission and compliance
